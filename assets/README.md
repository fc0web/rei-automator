# Assets

このディレクトリにはアプリケーションのアイコンやロゴを配置します。

## 必要なファイル

- `icon.png` - アプリケーションアイコン（512x512px推奨）
- `icon.ico` - Windowsアイコン（複数サイズを含む.icoファイル）

## 作成方法

1. 512x512pxのPNG画像を作成（Rei言語のロゴまたはRei Automator専用ロゴ）
2. オンラインツール（例：https://icoconvert.com/）で.icoファイルを生成
3. このディレクトリに配置

## 暫定対応

Phase 1では、アイコンファイルがなくてもアプリケーションは動作します。
Electronのデフォルトアイコンが表示されます。
